  1. Sea $f(x) = -x^3 - x $ y $ p_0 = -1$ . Use el m√©todo de Newton y de la Secante para encontrar $ p_2 $. ¬øSe podr√≠a usar \(p_0 = 0\)?
import numpy as np
import matplotlib.pyplot as plt
from scipy import optimize

f = lambda x: -x**3 - np.cos(x)

root = optimize.newton(f, -1, fprime=lambda x:-3*x**2 + np.sin(x))
print('La raiz de la funci√≥n con el m√©todo de Newton es: {}'.format(root))
root = optimize.newton(f, -1)
print('La raiz de la funci√≥n con el m√©todo de Secante es: {}'.format(root))
La raiz de la funci√≥n con el m√©todo de Newton es: -0.8654740331016144
La raiz de la funci√≥n con el m√©todo de Secante es: -0.8654740331016144

¬øSe puede usar \(p_0 = 0\)?
No, no se puede usar \(p_0 = 0\) para el m√©todo de Newton porque en ese punto la derivada de \(f(x)\) es:

\[ f'(0) = -3(0)^2 + \sin(0) = 0 \]

Esto hace que el denominador en la f√≥rmula de Newton se anule, y el m√©todo no puede avanzar. En cambio, el m√©todo de la secante podr√≠a usarse con \(p_0 = 0\) y \(p_1 \neq 0\).
